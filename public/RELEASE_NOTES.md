# InterviewPrep v1.3.0 — AI 面试备战助手

> 上传 JD + 简历，AI 自动押题、模拟面试、简历优化、面经采集。一站式面试准备工具。

---

## 📦 核心功能

### 📊 仪表盘
打开即见练习概览——总练习次数、模拟面试次数、平均得分一目了然。雷达图展示 STAR 完整性 / 量化程度 / 岗位匹配 / 表达结构 / 亮点突出五个维度的能力画像，热力图记录每日练习节奏。

### 🔍 分析 & 押题
粘贴岗位 JD 和你的简历，AI 自动完成：
- **JD 解析** — 提取公司、岗位、职责、要求
- **简历解析** — 识别实习经历、项目经历、技能栈
- **差距分析** — 匹配点 / 优势点 / 薄弱点一目了然
- **五类题型生成** — 行为面试、专业能力、项目深挖、压力测试、HR 面
- **知识库增强** — 自动从本地知识库补充相关题目

支持 Boss直聘 / 51job / LinkedIn 等岗位链接一键扒取。快速模式下仅生成 3 类核心题型，速度提升 5 倍。

### 💪 单题练习
从押题清单中任意选题，输入你的回答，AI 即刻评估：
- 五维深度评分（STAR 完整性、量化程度、岗位匹配、表达结构、亮点突出）
- 逐句点评（✅ 亮点 / ⚠️ 改进点，精确到引用原文）
- 改进版参考 + 关键改进点汇总
- 高分回答可一键存入话术库
- 练习历史自动保存，折叠面板回顾每次练习的题目、回答、评分和改进

### 📡 面经采集
独立搜索小红书面经，AI 提取真实面试题：
- **双通道采集** — 文字提取 + 截图 OCR（tesseract.js 中英文识别）
- **三阶段 LLM 结构化** — 粗提取所有候选题目 → 分类标签（行为/技术/项目/压力/HR）→ 增强输出（回答要点、技巧、考察意图）
- SSE 实时进度推送，OCR 语言包下载/识别进度可见
- 支持手动粘贴小红书链接批量抓取
- 结果可逐题 / 批量加入单题练习和真题库
- 一键同步到押题清单

### 🎤 全真模拟面试
AI 扮演面试官，从自我介绍开始，多轮追问：
- 智能追问 — 回答不充分时自动深挖
- 多阶段面试 — 行为面 → 专业面 → 项目面 → 压力面
- 优雅收尾 — 结束后自动生成五维雷达图 + 逐题点评报告
- 面试历史回顾 — 记录每次面试的日期、岗位、得分

### 📄 简历优化
AI 逐段分析你的简历，针对目标 JD 给出优化建议：
- 原文 vs 优化后 左右对照
- 一段一句优化理由
- 自动生成一句话 Elevator Pitch + 1 分钟自我介绍脚本

### 📚 真题库
面经采集的所有题目自动归档到真题库：
- 按公司 / 岗位 / 题型筛选
- 收藏高频题目
- 点击「练习」按钮直接跳转单题练习

### 🌙 暗色模式
Navbar 右上角一键切换暗色 / 亮色主题，跟随系统偏好，偏好记忆到本地。

### 📥 一键导出
- 押题清单 → Markdown / DOCX
- 练习历史 → Markdown
- 面试报告 → Markdown / DOCX

## 🚀 快速开始

### 方式一：直接运行（无需 Electron）

```bash
npm install
npm start
# 打开 http://localhost:3456
```

### 方式二：Electron 桌面应用

```bash
npm install
npm run electron
```

### 方式三：Windows 安装包

下载 Release 页面的 `InterviewPrep Setup 1.3.0.exe`，双击安装即可。

---

## ⚙️ AI 供应商配置

首次使用需配置 AI 供应商：

1. 点击 Navbar 右侧 ⚙️ 设置按钮
2. 选择供应商（支持 OpenAI / DeepSeek / 自定义 OpenAI-Compatible）
3. 填入 API Key + Base URL
4. 点击「测试连接」→ 保存

支持多供应商自动故障转移（DeepSeek 挂了自动切 OpenAI）。

## 🛠 技术栈

- **后端**: Node.js + Express + ai-provider-kit
- **前端**: 原生 HTML/CSS/JS（零框架，极速加载）
- **桌面端**: Electron
- **AI**: LLM（支持 OpenAI / DeepSeek / 任意 OpenAI-Compatible API）
- **OCR**: tesseract.js
- **文档解析**: mammoth (.docx) / pdf-parse (.pdf)
- **导出**: docx 库
